Custom Portfolio Website

Overview

This portfolio website is a testament to my dedication, creativity, and technical expertise in the world of computing. After experimenting with platforms like WordPress and Wix, I transitioned to a fully customized and self-hosted website, built entirely from scratch. Every aspect—ranging from the images and animations to the rocket models featured in the loading screens—was designed and developed by me. This website not only showcases my technical capabilities but also reflects my commitment to delivering unique and immersive experiences.

Custom Code & Interactive Animations

I hand-coded the website using HTML and JavaScript, ensuring complete control over structure and functionality. The interactive animations, including the dynamic rocket launch sequence on the loading screen, are powered by custom JavaScript. These animations were carefully crafted to enhance user engagement while maintaining seamless performance across devices.

Styling with CSS & Tailwind

To achieve precise and responsive styling, I combined traditional CSS stylesheets with the utility-first Tailwind CSS framework. This hybrid approach allowed me to fine-tune the visual presentation while maintaining a clean and maintainable codebase. I implemented complex design features such as a pulsating glow effect, smooth page transitions, and responsive layouts that adapt gracefully to different screen sizes.

Art & Multimedia Design

I personally created and edited all visual assets using Krita (shout-out to Krita, an amazing open-source digital painting software). This includes custom illustrations, image editing, and 3D rocket models for the animated sequences. Additionally, I recorded and edited all of the videos you see featured on the website, ensuring a cohesive visual theme throughout the site. Each visual element was designed to reinforce the website's narrative and highlight my multifaceted skill set.

It's on Github!

The entire development process was tracked and managed using GitHub for version control. This allowed me to document every stage of the website's evolution—from the initial codebase to the final polished product. GitHub enabled efficient tracking of changes, maintaining code integrity, and creating a transparent record of my work. The full project history and current project files are available on my GitHub repository, showcasing months of consistent development and iteration.
